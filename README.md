# Sophos
Chat IA em aprimoramento.

### Tecnologias
* Python
* Machine Learning

### Sobre

>Sophos é um Chat IA que acessa a internet para definir melhor suas respostas antes de apresentar ao usuário. Possui um gerador de imagens a partir de prompts.<br>Sophos foi baseado a partir do Gemini e Chat GPT.
